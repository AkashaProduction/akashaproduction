# Documentation Utilisateur

## Navigation

- `Présentation` expose la vision globale d’Akasha Production et de ses sous-projets.
- `Nos solutions` détaille les formules de création web, hébergement, domaine et packs.
- `Contact` permet d’envoyer un message complet avec pièces jointes.
- `Commander` permet de composer une offre ou de demander un devis.
- `Mon compte` permet de retrouver les commandes via l’email client.

## Commander

- Choisissez une formule de création.
- Ajoutez une formule d’hébergement si besoin.
- Sélectionnez un sous-domaine offert ou un domaine personnalisé.
- Activez éventuellement le paiement en `3x sans frais` pour les offres tarifées compatibles.
- Si le projet est sur mesure, un questionnaire supplémentaire apparaît.
- Une fois validée, la commande est enregistrée. Si Stripe est configuré, le client est redirigé vers Checkout.

## Packs automatiques

- Vitrine + mutualisé annuel: `120 €`
- Complexe + mutualisé annuel: `550 €`
- Des promotions cohérentes sont également prévues pour d’autres combinaisons affichées ou non affichées.

## Contact

- Tous les champs saisis sont sauvegardés.
- Les formats de fichiers acceptés sont `DOCX`, `PDF`, `JPG`, `WEBP`.
- Une confirmation s’affiche directement sur la page après envoi.

## Mon compte

- Le client saisit son email pour retrouver ses commandes.
- Les achats, options et informations de souscription sont affichés.
- La zone technique avancée est réservée pour les futures fonctionnalités.

## Tickets support

- Le panel client permet de créer un ticket rattaché soit au `Service commercial`, soit au `Service technique`.
- Chaque ticket propose des sujets cohérents selon la catégorie choisie.
- Le client peut relier son ticket à une commande existante.
- Les réponses de modération apparaissent dans le fil du ticket côté client.

## Administration

- Un panel administrateur dédié permet la connexion du superadmin.
- Le service admin peut modifier le statut, la priorité et répondre directement aux tickets.
- Les réponses peuvent être envoyées par email si le SMTP est configuré.
